## 프로젝트 개요
  학원을 수료 했지만 Spring과 Java 웹 개발에 필요한 본질적인 지식 더 이해하고 싶어 Spring과 JSP를 활용한 토이 프로젝트를 진행하고자 하였고
 3인 팀으로 하여 '데이트팝'사이트를 클론으로 목표로 진행하였습니다.
